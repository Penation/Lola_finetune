# **LoLA: 基于 FLUX.2 DiT 与 Qwen3.5 的视觉-语言-动作模型**

LoLA 是一款专为高精度机器人操作设计的前沿 VLA（Vision-Language-Action）模型。它将强大的 **Qwen3.5** 视觉语言大模型与深度优化的、参数量约为 600M 的 **FLUX.2** 风格扩散 Transformer（DiT）完美结合。本模型完全兼容 Hugging Face 的 LeRobot 框架。

## **🌟 核心架构亮点**

### **1\. VLM 多尺度特征提取**

为了避免大语言模型最后一层严重的“下一个词预测（Next-token）”偏置，LoLA 创新性地抽取了 Qwen3.5 第 **8, 16, 24 层**的隐藏状态。这种多层级特征融合能够同时捕获浅层的物理空间属性与深层的复杂语义逻辑。

### **2\. 基于 empty\_token 的全局意图控制**

模型在 VLM 序列末尾强制拼接了一个 empty\_token。该 Token 负责在自注意力中汇聚全局的任务意图。提取后，它与扩散过程的 Timestep（时间步）进行深度融合，生成全局调制信号（Modulation），通过 **AdaLN** 直接控制 DiT 的特征缩放与平移。

### **3\. 动作分块 (Action Chunking) 与 MLP 编码器**

LoLA 将 20 维的复杂物理动作（包含 3D 平移、6D 旋转和 1D 夹爪状态）按照 10 步一组进行分块（Chunking）。通过带有 **LayerNorm** 的 MLP 编码器，有效消除了不同物理量纲带来的方差偏移，并捕获了动作轨迹中高度非线性的运动趋势。

### **4\. FLUX.2 单双流混合 DiT 架构**

其 \~600M 的 DiT 主干网络采用了最先进的混合结构：

* **双流阶段 (4 层):** 目标动作流（Target）与 上下文流（VLM 特征 \+ 历史动作）各自独立计算，确保物理模态与视觉语义模态在初步交互时不丢失自身的特征分布特性。  
* **单流阶段 (12 层):** 所有序列合并，进行深度的跨模态特征纠缠与联合自注意力计算。

### **5\. 多轴 RoPE 旋转位置编码与模态隔离**

为彻底解决单流阶段容易出现的“位置坍塌”问题，LoLA 引入了 4D 坐标系 (T, H, W, L) 的多轴 RoPE：

* T=0 分配给视觉与文本特征  
* T=1 分配给历史动作  
* T=2 分配给目标生成动作  
  这种绝对的正交隔离机制，确保了模型绝对不会将“历史的物理动量”与“未来的目标轨迹”在注意力机制中混淆。

## **🚀 训练策略：流匹配 (x-pred 预测，v-loss 监督)**

LoLA 采用了**流匹配 (Flow Matching)** 机制，并在目标函数上做出了极其硬核的优化：

* **网络输出:** 模型参数化为直接预测干净的动作本体 (x-pred)。在推理阶段，这种设计完美消除了欧拉积分（Euler Integration）在最后微小时间步的截断误差。  
* **损失监督:** 采用预测流速与真实流速的均方差计算损失 (v-loss)。这隐式地引入了 1/t^2 的 SNR 权重，使得模型在极低噪声阶段（即t->0时）受到极严厉的惩罚，从而极大提升了机械臂末端高精度抓取的成功率。

### **两阶段训练 (Two-Stage Training)**

在 configuration\_lola.py 中，LoLA 提供了极其灵活的训练配置支持：

1. **预热 (Warm-up):** 设置 train\_vlm \= False。冻结 Qwen3.5 权重，让 DiT 在绝对干净、稳定的多模态特征下快速收敛。  
2. **联合微调 (Joint Fine-Tuning):** 设置 train\_vlm \= True。解冻 VLM，使用专门设置的低学习率 (vlm\_lr) 进行端到端的梯度反传，让视觉特征彻底对齐物理世界的动态变化。
